Introducción a PHP

1. ¿Qué es PHP?

PHP es un lenguaje de programación que se usa sobre todo en páginas web.

Sirve para hacer que una página no sea estática, sino que pueda procesar datos, conectarse con bases de datos, enviar información, etc.

2. ¿Qué puede hacer PHP?

  • Generar contenido dinámico en páginas web.
  • Conectarse y trabajar con bases de datos (ejemplo: MySQL).
  • Recibir y procesar formularios.
  • Manejar sesiones y cookies.
  • Crear sistemas como: login, blogs, tiendas en línea, etc.

3. ¿Cómo instalar MAMP?

• Ve a la página oficial de MAMP.
• Descarga la versión para tu sistema (Windows o Mac).
• Instálalo como cualquier programa.
• MAMP incluye Apache, PHP y MySQL.

4. ¿Cómo instalar Visual Studio Code (VS Code)?

• Descarga desde la página oficial de Visual Studio Code.
• Instálalo en tu computadora.
• Es un editor de código ligero y muy usado.
• Puedes agregar extensiones (ejemplo: PHP Intelephense).

5. ¿Qué es una carpeta de trabajo?

Es la carpeta principal de tu proyecto donde guardas tus archivos (PHP, HTML, CSS, imágenes, etc.).

6. ¿Qué es una extensión de archivos?

  • .php = código PHP
  • .html = páginas web estáticas
  • .css = estilos
  • .js = JavaScript

7. ¿Cómo editar código PHP?

Como en el ejemplo solo podemos cambiar el texto de "Hola Mundo" y poner cualquier otro texto:

<?php
  echo "Hola Mundo";
?>

8. ¿Cómo guardar archivos?

En VS Code → Ctrl + S (o en Mac Cmd + S). Asegúrate de guardar con la extensión correcta.

9. ¿Cómo visualizar archivos?

solamente nos metemos a nuestra carpeta de trabajo y buscamos el nombre que le hayamos puesto a nuestro archivo

10. ¿Qué son las variables en PHP?

<?php
  $nombre = "Michell";
  $edad = 20;
  echo "Hola, me llamo $nombre y tengo $edad años.";
?>
